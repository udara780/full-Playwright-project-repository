# Singlish to Sinhala Transliteration Accuracy Testing

This repository contains the automation scripts for testing the accuracy of the Singlish-to-Sinhala chat translator as part of the IT3040 - ITPM Assignment 1.

## Project Overview
The main objective is to evaluate how accurately the Pixelssuite Chat Translator converts chat-style Singlish input into Sinhala output. This project automates 50 negative test cases.

## Prerequisites
- Python 3.10 or higher
- Microsoft Excel
